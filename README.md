# E-Commerce Sales Data Analysis using Python Libraries in Jupyter Notebook

# Project Overview

This project focuses on analyzing e-commerce sales data using Python libraries in Jupyter Notebook to uncover valuable business insights. The analysis includes data cleaning, preprocessing, exploratory data analysis (EDA), and visualization to understand sales performance, customer behavior, and product trends.

# Objective

The main objectives of this project are to:

Analyze e-commerce sales performance
Identify top-selling products and categories
Understand customer purchasing behavior
Discover sales trends and revenue patterns
Generate actionable business insights through data visualization

# Dataset Details

The dataset contains information related to:

Orders and sales
Products and categories
Customer details
Revenue and profit
Purchase trends

# Tools & Technologies
Python
Jupyter Notebook
Pandas – Data manipulation
NumPy – Numerical analysis
Matplotlib – Data visualization
Seaborn – Statistical visualization

# Key Analysis Performed
Data Cleaning & Preprocessing
Exploratory Data Analysis (EDA)
Sales Trend Analysis
Product Performance Analysis
Customer Behavior Analysis
Revenue & Profit Insights
Data Visualization using charts and graphs
Key Insights
Identified high-performing products and categories
Analyzed monthly and yearly sales trends
Discovered customer purchasing patterns
Highlighted factors affecting revenue and profit
